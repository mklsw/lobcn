#### Python中的标识符(identifier)

        Identifier is the name given to entities like  class,functuons,variables etc.In Python. It helps differentating one entity 
        to another.

#### Python 中的标识符书写规则：

1. 标识符可以是（a-z 或者（A-Z）或者(0-9) 或者 (_)的自由组合.

2. 标识符不能用数字开头

3. 保留的关键字不能用作标识符

4. 不能使用特殊符号，& @ 等

5  长度不限
