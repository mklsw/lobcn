## String 字符串 

字符串的表示方法为单引号，双引号和三引号，字符串创建以后是不可更改的. 


    ''
    ""
    '''
    """
    

### 字符串的转义

转义字符 | 描述 |
:-------:|:-------:|
 \ (在行尾时)|续行符号|
\\      |反斜杠符号 |
\'     | 单引号|
