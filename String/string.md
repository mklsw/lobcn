## String 字符串 

字符串的表示方法为单引号，双引号和三引号，字符串创建以后是不可更改的. 


    ''
    ""
    '''
    """
    

### 字符串的转义

1. 遇到特殊字符时，Python 用反斜杠"\" 转义字符 

    
    >>> 'doesn\'t'
    "doesn't"
        
    >>> "doesn't"
    "doesn't"
    
    >>> '"yes,"they said.'
    '"yes,"they said.'
    
    >>> "\"yes,\"they said."
    '"yes,"they said.'
